Bistutu空教室查询
===================

安卓版本

## 项目简介

本软件适用于北京信息科技大学空教室查询，开发此软件是为了能够帮助小伙伴们迅速找到空教室。

## 客户端使用到的第三方库

```
1. Gradle 开发框架
2. okhttp3 网络请求
3. Gson JSON数据解析
4. QMUI 界面框架
```

## 服务器端使用到的第三方库

```
1. Maven 开发框架
2. fastJSON JSON数据解析
```
